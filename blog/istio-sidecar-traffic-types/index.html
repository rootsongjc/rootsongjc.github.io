жң¬ж–Үе°Ҷеҗ‘дҪ еұ•зӨә Istio sidecar дёӯзҡ„е…ӯз§ҚжөҒйҮҸзұ»еһӢеҸҠе…¶ iptables и§„еҲҷпјҢд»ҘзӨәж„Ҹеӣҫзҡ„еҪўејҸеёҰдҪ дёҖи§Ҳе…¶е…ЁиІҢпјҢе…¶дёӯиҜҰз»ҶжҢҮеҮәдәҶи·Ҝз”ұе…·дҪ“дҪҝз”Ёзҡ„жҳҜ ISTIO_OUTPUT дёӯзҡ„е“ӘдёҖжқЎи§„еҲҷгҖӮ

Sidecar дёӯзҡ„ iptables жөҒйҮҸи·Ҝз”ұ

Sidecar дёӯзҡ„жөҒйҮҸеҸҜд»ҘеҲ’еҲҶдёәд»ҘдёӢеҮ зұ»пјҡ

  • иҝңзЁӢжңҚеҠЎи®ҝй—®жң¬ең°жңҚеҠЎпјҡRemote Pod -> Local Pod
  • жң¬ең°жңҚеҠЎи®ҝй—®иҝңзЁӢжңҚеҠЎпјҡLocal Pod -> Remote Pod
  • Prometheus жҠ“еҸ–жң¬ең°жңҚеҠЎзҡ„ metricsпјҡPrometheus -> Local Pod
  • жң¬ең° Pod жңҚеҠЎй—ҙзҡ„жөҒйҮҸпјҡLocal Pod -> Local Pod
  • Envoy еҶ…йғЁзҡ„иҝӣзЁӢй—ҙ TCP жөҒйҮҸ
  • Sidecar еҲ° Istiod зҡ„жөҒйҮҸ

дёӢйқўе°Ҷдҫқж¬Ўи§ЈйҮҠжҜҸдёӘеңәжҷҜдёӢ Sidecar еҶ…зҡ„ iptables и·Ҝз”ұи§„еҲҷгҖӮ

зұ»еһӢдёҖпјҡRemote Pod -> Local Pod

д»ҘдёӢжҳҜиҝңзЁӢжңҚеҠЎгҖҒеә”з”ЁжҲ–е®ўжҲ·з«Ҝи®ҝй—®ж•°жҚ®е№ійқўжң¬ең° Pod IP зҡ„ iptables и§„еҲҷгҖӮ

Remote Pod -> RREROUTING -> ISTIO_INBOUND -> ISTIO_IN_REDIRECT -> Envoy 15006пјҲInboundпјү-> OUTPUT -> ISTIO_OUTPUT RULE 1 -> POSTROUTING -> Local Pod

жҲ‘д»¬зңӢеҲ°жөҒйҮҸеҸӘз»ҸиҝҮдёҖж¬Ў Envoy 15006 Inbound з«ҜеҸЈгҖӮиҝҷз§ҚеңәжҷҜдёӢзҡ„ iptables и§„еҲҷзҡ„зӨәж„ҸеӣҫеҰӮдёӢгҖӮ

зұ»еһӢдәҢпјҡLocal Pod -> Remote Pod

д»ҘдёӢжҳҜжң¬ең° Pod IP и®ҝй—®иҝңзЁӢжңҚеҠЎз»ҸиҝҮзҡ„ iptables и§„еҲҷгҖӮ

Local Pod-> OUTPUT -> ISTIO_OUTPUT RULE 9 -> ISTIO_REDIRECT -> Envoy 15001 (Outbound)-> OUTPUT -> ISTIO_OUTPUT RULE 4 -> POSTROUTING -> Remote Pod

жҲ‘д»¬зңӢеҲ°жөҒйҮҸеҸӘз»ҸиҝҮ Envoy 15001 Outbound з«ҜеҸЈгҖӮ

д»ҘдёҠдёӨз§ҚеңәжҷҜдёӯзҡ„жөҒйҮҸйғҪеҸӘз»ҸиҝҮдёҖж¬Ў EnvoyпјҢеӣ дёәиҜҘ Pod дёӯеҸӘжңүеҸ‘еҮәжҲ–жҺҘеҸ—иҜ·жұӮдёҖз§ҚеңәжҷҜеҸ‘з”ҹгҖӮ

зұ»еһӢдёүпјҡPrometheus -> Local Pod

Prometheus жҠ“еҸ–ж•°жҚ®е№ійқў metrics зҡ„жөҒйҮҸдёҚдјҡд№ҹж— йЎ»з»ҸиҝҮ Envoy д»ЈзҗҶгҖӮ

иҝҷдәӣжөҒйҮҸйҖҡиҝҮзҡ„ iptables и§„еҲҷеҰӮдёӢгҖӮ

Prometheus-> RREROUTING -> ISTIO_INBOUNDпјҲеҜ№зӣ®зҡ„ең°дёә 15020гҖҒ15090 з«ҜеҸЈжөҒйҮҸе°ҶиҪ¬еҲ° INPUTпјү-> INPUT -> Local Pod

иҝҷз§ҚеңәжҷҜдёӢзҡ„ iptables и§„еҲҷзҡ„зӨәж„ҸеӣҫеҰӮдёӢгҖӮ

зұ»еһӢеӣӣпјҡLocal Pod -> Local Pod

дёҖдёӘ Pod еҸҜиғҪеҗҢж—¶еӯҳеңЁдёӨдёӘжҲ–еӨҡдёӘжңҚеҠЎпјҢеҰӮжһң Local Pod и®ҝй—®зҡ„жңҚеҠЎд№ҹеңЁиҜҘеҪ“еүҚ Pod дёҠпјҢжөҒйҮҸдјҡдҫқж¬Ўз»ҸиҝҮ Envoy 15001 е’Ң Envoy 15006 з«ҜеҸЈжңҖеҗҺеҲ°иҫҫжң¬ең° Pod зҡ„жңҚеҠЎз«ҜеҸЈдёҠгҖӮ

иҝҷдәӣжөҒйҮҸйҖҡиҝҮзҡ„ iptables и§„еҲҷеҰӮдёӢгҖӮ

Local Pod-> OUTPUT -> ISTIO_OUTPUT RULE 9 -> ISTIO_REDIRECT -> Envoy 15001пјҲOutboundпјү-> OUTPUT -> ISTIO_OUTPUT RULE 2 -> ISTIO_IN_REDIRECT -> Envoy 15006пјҲInboundпјү-> OUTPUT -> ISTIO_OUTPUT RULE 1 -> POSTROUTING -> Local Pod

зұ»еһӢдә”пјҡEnvoy еҶ…йғЁзҡ„иҝӣзЁӢй—ҙ TCP жөҒйҮҸ

Envoy еҶ…йғЁиҝӣзЁӢзҡ„ UID е’Ң GID дёә 1337пјҢе®ғд»¬д№Ӣй—ҙзҡ„жөҒйҮҸе°ҶдҪҝз”Ё lo зҪ‘еҚЎпјҢдҪҝз”Ё localhost еҹҹеҗҚжқҘйҖҡдҝЎгҖӮ

иҝҷдәӣжөҒйҮҸйҖҡиҝҮзҡ„ iptables и§„еҲҷеҰӮдёӢгҖӮ

Envoy иҝӣзЁӢпјҲLocalhostпјү -> OUTPUT -> ISTIO_OUTPUT RULE 8 -> POSTROUTING -> Envoy иҝӣзЁӢпјҲLocalhostпјү

зұ»еһӢе…ӯпјҡSidecar еҲ° Istiod зҡ„жөҒйҮҸ

Sidecar йңҖиҰҒи®ҝй—® Istiod д»ҘеҗҢжӯҘй…ҚзҪ®пјҢpilot-agent иҝӣзЁӢдјҡеҗ‘ Istiod еҸ‘йҖҒиҜ·жұӮпјҢд»ҘеҗҢжӯҘй…ҚзҪ®гҖӮ

иҝҷдәӣжөҒйҮҸйҖҡиҝҮзҡ„ iptables и§„еҲҷеҰӮдёӢгҖӮ

pilot-agent иҝӣзЁӢ -> OUTPUT -> Istio_OUTPUT RULE 9 -> Envoy 15001 (Outbound Handler) -> OUTPUT -> ISTIO_OUTPUT RULE 4 -> POSTROUTING -> Istiod
