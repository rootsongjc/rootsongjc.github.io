Cloud Native, Open Source and Community

Jimmy Song is the founder of Cloud Native Community and has authored or translated several books on cloud native since 2016. He is committed to promoting cloud native technologies and trying to build a developer ecosystem based on containers, Kubernetes, Service Mesh, and Serverless in order to help Chinese developers get involved in the global cloud native trend. Cloud Native Community he’s assembled covers almost all the stalwarts of cloud native in China, join the community and walk alongside them!

Start your Cloud Native journey

Cloud Native is a behavior and design philosophy. At its essence, any behavior or approach that improves resource utilization and application delivery efficiency in the cloud is Cloud Native. The history of cloud computing is a history of Cloud Native biochemistry. Kubernetes kicked off the prelude of cloud native 1.0, and the emergence of Istio, a service mesh, led the way for microservices in the post-Kubernetes era. The re-emergence of serverless makes cloud native move from the infrastructure layer to the application architecture layer, and we are now in a new era of cloud native.
