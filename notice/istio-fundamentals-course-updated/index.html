公告 Istio 基础教程 2025 年版本更新

我在 Tetrate Academy 上主导更新的 Istio Fundamentals 课程现已全新上线，全面支持 Istio v1 API、Gateway API 和 Ambient 模式，欢迎免费学习。

  • 2025/05/09
  • 2 分钟
  • 464 字

我最近完成了对 Tetrate Academy 上 Istio Fundamentals 课程 的一次重大更新。这个课程最初于 2021 年上线，并在 2023 年做过一次小规模修订。此次是该课程发布以来最全面的一次更新，覆盖内容、版本、技术栈、视频与结构优化。

本次更新在保留原有课程 大纲结构 和学习节奏的基础上，全面对内容进行了重构与升级，以适配 Istio 社区的发展趋势。

更新亮点如下：

  • 全面升级至 Istio 1.24.3：课程原版本基于 Istio 1.17.2，现已更新为 2025 年 2 月发布的 Istio 1.24.3。
  • 所有资源切换为稳定版 v1 API：不再使用 beta API，更贴近实际生产环境配置。
  • 新增 Ambient 模式模块：涵盖架构原理、数据路径与配置方式。
  • 引入 Kubernetes Gateway API：作为服务暴露的现代替代方案。
  • 课程视频全部重录：取消讲师出镜画面，内容更专注、节奏更紧凑。
  • 移除 getmesh 工具：统一使用 istioctl 和 Helm。
  • 更新真实案例模块：与新版本 Istio 和 Kubernetes 保持一致。
  • 新增 TIS Overview 模块：简要介绍 Tetrate Istio Subscription 的支持能力。
  • 测验题优化：删除过时题目，增加新版内容相关测试。
  • 整体内容排版与节奏优化：学习路径更加清晰流畅。

本课程完全免费，适合想系统学习 Istio 或了解其最新架构演进（如 Ambient 和 Gateway API）的云原生开发者、运维工程师与架构师。
