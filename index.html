云原生社区

云原生社区由 Jimmy 和他的伙伴们发起于 2020 年 2 月，经过几个月时间的筹备，现在我们要向全世界宣布，云原生社区正式成立了！这是一个有温度的开源社区，我们希望借助开源打破公司的边界，我们关注技术人的成长，促进社区之间的交流。让我们一起关注开源，回馈开源。

开启云原生之旅

云的之势不可阻挡，开发者应该顺应云原生，积极接纳云原生，让云为我所用，最大限度发挥云的优势，又快又好的实现开发者的业务价值。让我们先从了解 Kubernetes 开始。

开始

容器

云原生应用的最小的构建单元，一次构建，到处运行。

Kubernetes

云原生操作系统，声明式 API 是它的核心，Operator 操控万物。

Service Mesh

服务网格作为云原生网络基础设施，负责服务间通信的一切方面。

Serverless

可预见时间内的云原生的终极形式，真正的按需使用和按量付费。
