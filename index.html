关注领域

容器作为云原生应用的标准交付物和最小的构建单元，一次构建，到处运行。

Kubernetes 是云原生时代的操作系统，以不可变基础设施为设计的理念，以声明式 API 为核心，Operator 扩展的引擎。

服务网格作为云原生网络基础设施，负责服务间通信的一切方面，旨在构建零信任网络。

可预见时间内的云原生的终极形式，真正的按需使用和按量付费。
